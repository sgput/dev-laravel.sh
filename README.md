# Personal Local Laravel Development Setup

## Description: Opens Google Chrome, Virtual Box and Sublime Text. Changes into appropriate directories in iTerm tabs, ssh's into Homestead VM.

Dependencies include:
  * Laravel 5.2
  * Homestead
  * VirtualBox
  * Sublime Text (3)
  * Google Chrome
  * iTerm2
